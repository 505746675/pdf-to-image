# PDF转图片工具 - Windows版

一个功能强大的PDF转图片转换工具，支持单文件和批量转换，提供直观的图形界面。

## 功能特点

- ✅ 支持单PDF文件转换
- ✅ 支持批量PDF文件转换
- ✅ 多种输出格式（PNG、JPEG、BMP、TIFF）
- ✅ 可调节DPI设置（72-600）
- ✅ 自定义输出文件名前缀
- ✅ 实时转换进度显示
- ✅ 详细的转换日志
- ✅ 自动打开输出文件夹

## 系统要求

- Windows 7/8/10/11
- Python 3.7+（如果运行源代码）

## 安装使用

### 方法一：使用预编译版本（推荐）

1. 从 [Releases](https://github.com/yourusername/pdf-to-image-tool/releases) 页面下载最新版本的 `PDF转图片工具.exe`
2. 直接运行可执行文件，无需安装其他依赖

### 方法二：从源代码运行

1. 克隆或下载本项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
